## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Usage

1. Fill in your personal information in the form on the left
2. Add your work experience, education, skills, and projects
3. Watch the live preview update on the right
4. Click "Download PDF" to export your resume
